Private

  • Private(override?: string | boolean): (Anonymous function)
    • Defined in core/@model/private.ts:36

    Decorates a Model property and signals SakuraApi to not include that property when generating a json representation of the model.

    Example

    @Model()
class User {
   @Private()
   password: string = '';
}

    It's possible to conditionally make a property private by passing in the name (as a string) of a method that returns true or false (true meaning override the privacy setting) or by directly passing in a boolean value. The IJsonOptions.context will be passed as the first parameter of the method.

    Example of Override

    @Model()
class User {
   @Private('hasSsnAccess')
   ssn: string = '';

   hasSsnAccess(context:string) {
     return someAclChecker(this, 'ssn_access');
   }
}

    Parameters

    • Optional override: string | boolean

    Returns (Anonymous function)
